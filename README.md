
# NLW eSports

Um sistema para encontrar jogadores e fazer publicações procurando grupo para jogar.


## Referência

 - [Projeto Figma](https://www.figma.com/file/WEoUtFHvZHevnAJbBZpyNP/NLW-eSports-(Community)?node-id=0%3A1)


## Deploy

Para fazer o deploy desse projeto rode

```bash
(Server)
  ..\NLW-eSports\server> npm run dev
```

```bash
(Mobile) instalar o app Expo Go
  ..\NLW-eSports\mobile> expo start
```

```bash
(Web)
  ..\NLW-eSports\web> npm run dev
```

**Front-end:** React, Radix, TailwindCSS, Axios e Phosphor

**Back-end:** Node e  Prisma

**Mobile:** React Native e Expo
